# Naresh Kumar D - Portfolio AI Context

## Project Purpose
A personal portfolio website showcasing the experience, projects, skills, and certifications of Naresh Kumar D, a Full Stack Software Engineer based in Chennai, Tamil Nadu, India.

## Folder Structure
- `app/` - React 18 + Vite frontend with Tailwind CSS and Framer Motion.
- `api/` - FastAPI backend for handling contact requests and resume delivery.
- `docs/` - Developer documentation and AI context files.

## Conventions & Rules
- **Styling**: Tailwind CSS with custom design system (dark navy/midnight black base, electric blue/cyan accents). Uses dark mode by default. Minimalist, premium, glassmorphism design.
- **Component Patterns**: Functional components in React using hooks (`useContext`, `useEffect`).
- **Icons**: `lucide-react` and `react-icons`.
- **Animations**: Standardize on `framer-motion` for scroll-triggered and entrance animations.
- **Data Flow**: Contact form POSTs to FastAPI backend to trigger SMTP email.
